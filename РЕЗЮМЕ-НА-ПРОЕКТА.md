## ТЕМА: Отзивчиви уеб сайтове за домашни любимци

### КРАТКО ОПИСАНИЕ:

Разработени са два отделни single-page уеб сайта с тематика свързана с грижата за домашни любимци. Първият сайт "Paws & Rest Pet Hotel" е изграден изцяло "от нула" с чист HTML5 и CSS3, докато вторият "Happy Paws Pet Shop" използва Bootstrap 5.3.2 фреймуърк.

Сайт "от нула" демонстрира владеене на CSS Grid за 3-колонен layout при ширина над 1200px, три вида навигация (хоризонтална залепваща, хоризонтална втора и вертикална в страничната лента), както и пълна отзивчивост при всички стандартни breakpoints. Приложени са семантични HTML5 тагове (`<header>`, `<nav>`, `<article>`, `<aside>`, `<section>`, `<footer>`) и разнообразни CSS свойства за позициониране (position: sticky, display: grid/flex, transitions).

Bootstrap сайтът показва уменията за работа с готови компоненти и 12-колонна grid система, като персонализирането е сведено до минимум - само 47 реда custom CSS за hero gradient и card hover ефекти. Използвани са Bootstrap navbar, cards, forms и utility класове за spacing и colors.

Галериите на двата сайта съдържат placeholder изображения от Unsplash. Двата проекта са напълно responsive с плавна навигация между секциите и оптимизирана видимост на заглавията при скролиране.
